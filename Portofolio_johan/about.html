About Me



Perkenalkan, saya Johan Edwin Tambunan, mahasiswa Universitas Mercu Buana jurusan Teknik Infomarika. Saya memiliki minat yang mendalam dalam analisis data, machine learning, visualisasi informasi, Atau bisa juga mengelola website, dan bisa juga menangani UI/UX. Tapi saya Berfokus Kedalam bidang data karena saya meyakini bahwa setiap data menyimpan informasi penting, dan peran seorang analis adalah mengekstraksi insight yang dapat mendukung pengambilan keputusan yang tepat.Selama menempuh pendidikan dan mengikuti berbagai proyek, saya telah mengembangkan kemampuan dalam:

- Analisis Data & Statistik menggunakan Python (Pandas, NumPy, Matplotlib)
- Manajemen dan Query Data menggunakan SQLVisualisasi Data & Dashboarding dengan Power BI dan Tableau
- Pemodelan Machine Learning Dasar, termasuk regresi dan klasifikasi
- Pengelolaan Website menggunakan CMS seperti WordPress dan dasar-dasar HTML/CSS
- Desain UI/UX menggunakan canva untuk menciptakan pengalaman pengguna yang intuitif dan menarik.
Fokus utama saya adalah mengubah data mentah menjadi informasi yang bernilai dan actionable, baik untuk keperluan akademik maupun bisnis. Saya berkomitmen untuk bekerja secara teliti, analitis, dan berbasis data untuk memberikan solusi yang optimal.

Saya terbuka untuk peluang magang atau posisi junior Data Analyst, di mana saya dapat menerapkan kemampuan analisis dan kontribusi dalam pengambilan keputusan berbasis data.
